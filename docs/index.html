Rally

This is a tool for building real-life rallies for eg church youth groups, or really anyone who wants to have a bit of fun with friends!

The idea is that an organiser decides on several points of interest around their town that people must visit, and get a photo at each point as evidence.

Clues are provided to take people from point to point. If a clue is too cryptic, in a few minutes an easier clue will be provided.

First team to the end point wins!

Drive carefully! Obviously no liability can be taken.

To start with, an organiser uses the 'builder' below to choose all the points, by clicking on the map, then clicking on the various text fields to enter the hints. This will only work on a desktop computer. Then he/she will click the 'publish' button, which will create a long link with the rally details. The intention is that this link will be minified using tinyurl. The resulting tinyurl link is to be shared with contestants at the start of the competition via QR, or by sharing the link in a group chat. Contestants will then open the link on a navigator's phone (NOT the driver!), which will monitor their GPS location to provide hints as appropriate. Contestants will need location permissions enabled for Safari/Chrome; it is worth ensuring everyone has this 'ready to go' before the competition. Good luck, have fun, stay safe, God bless!
